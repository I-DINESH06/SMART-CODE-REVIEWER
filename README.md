# 🤖 Smart Code Reviewer

An AI-powered code review application built using Flask and Google's Gemini API.

## Features

- AI Code Review
- Bug Detection
- Best Practice Suggestions
- Time Complexity Analysis
- Space Complexity Analysis
- Optimized Code Generation
- Supports Python, C, C++, Java and JavaScript

## Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript
- Google Gemini API

## Run

```bash
pip install -r requirements.txt
python app.py
```
